Akatsuki Shop es un ecommerce desarrollado con Laravel 12, PHP 8.2 y Tailwind CSS. El proyecto simula el funcionamiento de plataformas como Shopify e integra el flujo completo de compra utilizando el SDK oficial de Transbank Webpay Plus.

Estado del proyecto
80% completado

Tecnologías

Laravel 12 PHP 8.2 Blade Tailwind CSS SQLite Eloquent ORM Laravel Sessions Migrations Seeders Webpay Plus SDK Transbank

Funcionalidades implementadas

  • Catálogo de productos.
  • Vista individual de productos.
  • Reviews de clientes.
  • Carrito lateral tipo Shopify.
  • Agregar, editar y eliminar productos del carrito.
  • Checkout completo.
  • Integración con Webpay Plus Sandbox.
  • Confirmación automática de pagos.
  • Persistencia mediante sesiones Laravel.
  • Diseño responsive.

Próximas funcionalidades

  • Panel Administrativo.
  • CRUD de productos.

Lo que demuestra este proyecto

Laravel Arquitectura MVC Ecommerce Webpay Plus Transbank SDK Integración de APIs Blade Tailwind CSS Backend Responsive Design Eloquent ORM

Objetivo del proyecto

Este proyecto busca demostrar mis conocimientos en el desarrollo de aplicaciones ecommerce con Laravel, implementando un flujo completo de compra, integración con Webpay Plus mediante el SDK oficial de Transbank, arquitectura MVC y buenas prácticas de desarrollo, simulando el funcionamiento de plataformas modernas como Shopify.
